# reverse polish notation calculator in ruby

#[current version](https://repl.it/D3eA/43)

* takes 1 line of input in the form of "1 23  32 + 33 / 13 - / "

* working symbols: 
    * +
    * -
    * *
    * /
    * %
    * \**
